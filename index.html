To those interested in my philosophy, my why:

Every capability that counts as intelligence eventually turns inward. A model that can write code can write the code that trains the next model. A model that can do research can point that research at itself. This is no longer hypothetical. Roughly a quarter of the code inside frontier labs is now written by their own models. DeepMind's AlphaEvolve used one such model to discover a matrix-multiplication algorithm faster than the best humans had found in fifty years, and to reclaim compute across Google's data centers, the very substrate the next model trains on. OpenAI has models writing plain-English explanations of the individual neurons inside other models. Once the loop closes, progress stops scaling with the compute that can be bought and starts scaling with the compute the system chooses to spend. Call the result superintelligence; the mechanism underneath is humbler and more unsettling. It is recursive self-improvement, an optimizer pointed at the optimizer.

Biology is the first domain that loop will touch at full strength. In 2022 a group of drug-discovery researchers inverted the reward on one of their own generative models; in under six hours it proposed some forty thousand candidate toxins, rediscovering the nerve agent VX and inventing others no one had ever catalogued. The same class of models let AlphaFold map more than two hundred million protein structures, a century of crystallography compressed into a single release, while genomic models like Evo now read and write DNA directly. In most fields a capable model is an accelerant. In biology it is dual-use by default; the tool that designs a vaccine and the tool that designs a threat are the same weights with a different prompt. This is where a capability overhang meets the least forgiving consequences, and where a small alignment error is measured not in dollars or downtime but in lives.

So the answer is not to slow the loop; that race is already lost. The answer is to make it legible. Mechanistic interpretability is the closest thing to a microscope for cognition that currently exists, and it has started to work. When researchers reverse-engineered a small network trained only on modular arithmetic, they found it had quietly taught itself the discrete Fourier transform. A model fed nothing but Othello moves turned out to maintain a full board in its activations, a world model no one put there. Anthropic's sparse autoencoders pull millions of monosemantic features out of superposition, among them a single direction inside Claude that fires on the Golden Gate Bridge and neighboring ones for deception and sycophancy. Linear probes can read a model's internal state and flag a chain of thought that has begun to lie before it finishes the sentence, and unlearning methods scored on benchmarks like WMDP can excise the knowledge to synthesize a select agent instead of merely teaching the model to refuse. If a mind is going to improve itself, the priority is to read what it values while that mind still fits in human hands.

Anthropic's own Claude Mythos model card states this plainly. Mythos is the best-aligned model the lab has shipped, yet the report judges it the one posing the greatest alignment risk, because its capabilities in autonomous research and engineering now far outstrip any earlier model. As a system grows more capable, more autonomous, and more heavily relied upon, the average failure grows rarer while the failures that slip through grow far more consequential, which is why the automated research loop is where the stakes climb fastest.

The open question is whether a system can be understood faster than it can surprise. Today the honest answer is not yet. The work is to read the mind before it learns to read itself. It is the most important problem there is, and, unlike most things worth wanting, it is a tractable one.

TL;DR: Recursive self-improvement is the precursor to superintelligence. AI × bio and AI safety are the problems that compound faster than anyone can afford to ignore, and interpretability is the best available means of keeping them legible.
